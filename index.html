Sobre a Reciclagem

A reciclagem é um meio de preservar o meio ambiente, para que não ocorra a sua extinção.

Reciclar significa diminuir a quantidade de resíduos provenientes dos produtos consumidos pelo ser humano.

Fatos Curiosos

Quanto tempo demora para o plastico se degradar

Para se decompor leva cerca de 450 anos.

vidro

Quanto tempo demora para o vidro se degradar

Para que haja decomposição leva cerca de 1.000.000 de anos.

aluminio

Quanto tempo demora para o aluminio se degradar

Para que ocorra decomposição leva cerca de 200 a 500 anos.

papel

Quanto tempo demora para o papel se degradar

Para que ocorra decomposição leva cerca de 3 a 6 meses.
